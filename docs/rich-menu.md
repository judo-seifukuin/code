# リッチメニュー設定手順（Phase 4: 本番稼働）

LINE 公式アカウント Manager で姿勢チェックアプリをリッチメニューから起動できるようにする手順。

---

## 1. 前提

- LINE 公式アカウント開設済み
- LIFF アプリのエンドポイント URL が公開済み（Phase 0 セットアップ完了）
- LIFF ID を取得済み

---

## 2. リッチメニュー画像の準備

- 推奨サイズ: **2500 × 1686 px**（リッチメニュー大）または **2500 × 843 px**（小）
- ファイル形式: JPG / PNG
- ファイルサイズ: 1 MB 以下
- 「姿勢チェック」エリアにはアイコン＋テキストで誘導が分かるデザインに。

簡易例（小サイズ・2分割）：

```
+-----------------------------+-----------------------------+
|       📸 姿勢チェック       |       💬 お問い合わせ      |
+-----------------------------+-----------------------------+
```

---

## 3. LINE Official Account Manager で設定

1. https://manager.line.biz/ にログインし、対象アカウントを開く。
2. 左メニュー `トークルーム管理 → リッチメニュー` を開く。
3. `作成` をクリック。
   - タイトル: 「メインメニュー」
   - 表示期間: 適切な期間
   - メニューバーのテキスト: 「メニュー」
   - デフォルト表示: ON
4. テンプレートを選択（例: 小サイズの2分割）。
5. 背景画像をアップロード。
6. アクション設定:
   - 領域 A（姿勢チェック）:
     - タイプ: **リンク**
     - URL: `https://liff.line.me/<LIFF_ID>`
     - アクションラベル: 「姿勢チェック」
   - 領域 B（お問い合わせ）:
     - タイプ: テキスト / リンク等、運用に合わせて設定
7. `保存` → `表示する` に切り替えて公開。

---

## 4. 動作確認

1. テスト用アカウントで友だち追加。
2. トーク画面下部にリッチメニューが表示されることを確認。
3. 「姿勢チェック」をタップ → LIFF が起動 → 同意画面が出ること。

---

## 5. 多バリエーション運用（任意）

来院段階ごとに表示メニューを切り替えたい場合：

- 新規患者: 姿勢チェック大きめ + 予約導線
- 継続患者: 姿勢チェック + 履歴閲覧導線 + クーポン

リッチメニューを複数作成し、Messaging API 経由でユーザーごとに切替できる（要開発）。MVP では単一メニューで開始することを推奨。

---

## 6. 案内メッセージ（あいさつメッセージ）

`ホーム → あいさつメッセージ` で友だち追加時の自動メッセージを設定：

```
{院名}へようこそ！

📸 「姿勢チェック」を使うと、ご自身の姿勢の傾向を可視化できます。
画面下のメニューからお試しください。

※ 撮影画像はあなたのスマホ内だけで処理され、外部には送信されません。
```
